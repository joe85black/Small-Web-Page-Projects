# Small Web Page Projects

A portfolio of small, self-contained HTML/CSS practice projects — landing pages, forms, documentation pages, and CSS-only art/illustration exercises. Each project lives in its own folder with an `index.html` (+ `styles.css` where applicable) and a short README describing what it demonstrates.

No build tools or dependencies — open any project's `index.html` directly in a browser to view it.
